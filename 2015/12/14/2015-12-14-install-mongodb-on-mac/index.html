在Mac上安装MongoDB

在Mac上安装MongoDB有两种方法，也是官方给出的两种方法。

第一种，直接使用homebrew安装

$ brew install mongodb

但是，这种方法可能会存在问题。在安装的过程中，到从Google上下载go--tools那一步的时候，总是会失败(总之我没成功过，尽管可以通过浏览器访问到)。所以，也就有了手动安装的方法。

第二种，就是手动安装，官方文档里，关于手动安装的过程，我觉得少了一个关键点就是，没有说明应该把MongoDB安装在哪儿。

所以，这里总结一下手动安装的方法。

首先，去MongoDB官网下载MongoDB的压缩包，也可以使用命令，这里，我们将其下载到~/Downloads里面。

$ curl -O https://fastdl.mongodb.org/osx/mongodb-osx-x86_64-3.2.0.tgz

第二、解压

$ tar xzf mongodb-osx-x86_64-3.2.0.tgz

第三、安装，即将这个文件夹复制到/usr/local目录下

先重命名

$ mv mongodb-osx-x86_64-3.2.0 mongodb

然后，复制(或者移动)到/usr/local目录下(这里要加上sudo)

$ sudo mv mongodb /usr/local/

第四、创建MongoDB的数据库存放文件夹

$ sudo mkdir -p /data/db

然后，为这个文件夹指定用户权限，这里就指定你自己

$ sudo chown User /data/db

第五、配置环境变量

用vim编辑器打开.bash_profile文件

$ sudo vim ~/.bash_profile

然后，将下面的两行代码加入到里面

export MONGO_PATH="/usr/local/mongodb"
export PATH=$PATH:$MONGO_PATH/bin

wq保存之后，使用以下命令让.bash_profile生效

$ source ~/.bash_profile

然后，MongoDB就安装好了。

$ mongo --version 可以查看数据库版本。

$ mongod 可以启动数据库
